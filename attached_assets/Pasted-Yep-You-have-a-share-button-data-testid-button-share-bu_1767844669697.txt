## 1) Add a small “Share sheet” UI (tap link to copy + copy icon)

```tsx

```

Add state near the top of `VenturrDetail`:

```tsx
const [shareOpen, setShareOpen] = useState(false);
const [copied, setCopied] = useState(false);
```

Add helpers:

```tsx
const shareUrl = collection.shareSlug
  ? `${window.location.origin}/v/${collection.shareSlug}`
  : `${window.location.origin}/venturr/${collectionId}`; // fallback

async function copyToClipboard(text: string) {
  try {
    await navigator.clipboard.writeText(text);
  } catch {
    // iOS/Safari fallback
    const ta = document.createElement('textarea');
    ta.value = text;
    ta.style.position = 'fixed';
    ta.style.left = '-9999px';
    ta.style.top = '0';
    document.body.appendChild(ta);
    ta.focus();
    ta.select();
    document.execCommand('copy');
    document.body.removeChild(ta);
  }
}
```

Now make your share button open the sheet:

## 2) Add the share UI (Radix Dialog — iOS friendly)

If you already use Radix via shadcn, import Dialog:

```tsx
import {
  Dialog,
  DialogContent,
  DialogHeader,
  DialogTitle,
} from '@/components/ui/dialog';
```

Then add this near the bottom of your JSX (before `</div>`):

```tsx
<Dialog open={shareOpen} onOpenChange={setShareOpen}>
  <DialogContent className="sm:max-w-md">
    <DialogHeader>
      <DialogTitle>Share Venturr</DialogTitle>
    </DialogHeader>

    <div className="space-y-3">
      <p className="text-sm text-muted-foreground">
        Tap the link to copy (iPhone), or use the copy button (desktop).
      </p>

      <div className="flex items-center gap-2">
        {/* Tappable link pill */}
        <button
          onClick={async () => {
            await copyToClipboard(shareUrl);
            setCopied(true);
            toast.success('Link copied');
            setTimeout(() => setCopied(false), 1200);
          }}
          className="flex-1 text-left px-3 py-2 rounded-lg border border-border bg-muted/40 hover:bg-muted transition-colors"
        >
          <div className="text-xs text-muted-foreground">Share link</div>
          <div className="font-mono text-sm truncate">{shareUrl}</div>
        </button>

        {/* Copy icon (desktop-friendly) */}
        <button
          onClick={async () => {
            await copyToClipboard(shareUrl);
            setCopied(true);
            toast.success('Link copied');
            setTimeout(() => setCopied(false), 1200);
          }}
          className="h-10 w-10 rounded-lg border border-border hover:bg-muted flex items-center justify-center"
          aria-label="Copy share link"
        >
          {copied ? <Check className="w-4 h-4" /> : <Copy className="w-4 h-4" />}
        </button>
      </div>
    </div>
  </DialogContent>
</Dialog>
```

This accomplishes:

* iPhone: big tappable pill → copies
* Desktop: copy icon → copies
* Plain short URL shown
